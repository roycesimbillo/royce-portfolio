# Professional E-Portfolio
A fully featured, modern, and responsive e-portfolio website built with HTML5, CSS3, and Vanilla JavaScript. This portfolio showcases professional web development and UI/UX design work with a focus on performance, accessibility, and user experience.

# Features
- **Fully Responsive Design** - Works seamlessly on mobile, tablet, and desktop devices
- **Dark/Light Mode Toggle** - User preference-based theme switching with localStorage persistence
- **Smooth Animations & Transitions** - Engage users with subtle, performant animations
- **SEO Optimized** - Meta tags, structured data, and semantic HTML for better search engine visibility
- **Web Accessibility** - WCAG 2.1 AA compliant with ARIA roles and semantic markup

# Pages
## Home Page
The Home Page features an eye-catching hero section with a strong value proposition, highlighted statistics, engaging benefit cards, featured projects, client testimonials, and a fully functional navigation and footer to create a professional and compelling first impression.

## About Page
The About Page presents a comprehensive personal bio, detailed technical skills with progress indicators, education and certifications, professional experience timeline, and a clear breakdown of expertise to showcase qualifications and capabilities.

## Projects Page
The Projects Page displays four detailed case studies, each including featured and secondary images, project descriptions, key features, technology stacks, measurable outcomes, and live demo and repository links, arranged in an alternating layout for visual appeal.

## Contact Page
The Contact Page provides complete contact information, social media links, a fully functional and validated contact form with real-time feedback, a strong call-to-action section, and a response time indicator to encourage communication and engagement.

# Responsive Breakpoints
- **Desktop**: 1024px and above
- **Tablet**: 768px - 1023px
- **Mobile**: Below 768px
- Mobile navigation adaptations
- Touch-friendly button sizes

# Security & Best Practices
- No sensitive data exposure
- Secure rel attributes on external links (noopener, noreferrer)
- Form validation to prevent invalid submissions
- Error handling for failed operations
- CORS-ready structure for future API integration

# Technologies Used
